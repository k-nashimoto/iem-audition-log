## 音の好み（4+1軸／これが選定の基準）

- **低域**：存在感はしっかり、ただし量は出しすぎない。締まって「グルーヴ」を生む＝量より質。boomy/ブーミーは苦手。
- **高域**：キラキラした輝き＋高い解像度。**ただし刺さらないことが絶対条件**（サ行・8〜10kHzのピークに敏感）。
- **中域/ボーカル**：生々しさと「艶」。emotional realism を重視。
- **空間**：**「包まれる没入感」が最重要の決定軸**（前方だけでなく上下・後方に回り込む3D的な没入）。映画サントラ・オーケストラの大編成スケール。
- **生音感**：楽器の実在感。2方向で捉える → (A)温度・実体・木質感／(B)空気・余韻・倍音のきらめき。
- **嫌うもの**：clinical coldness（無機質）、boomy bass、treble spike。

## よく聴くジャンル

J-pop / K-pop / 映画・ゲームOST / オーケストラ。

## 機材

- **リファレンスIEM**：Vision Ears VE10（所有）。**音数少なめの女性ボーカルが最高**＝これがVE10の担当領域。高感度で iFi アッテネーター併用、Effect Audio A×C ケーブル(4.4mm)。生音感は(A)温度・実体型。
- **2台目の最有力候補**：Noble Kronos（約¥693,000）。用途は**映画/オーケストラの「包まれる没入」**。VE10と補完関係。生音感は(B)空気・余韻型。中域の"艶"はVE10ほど濃くないが、2台体制なので艶はVE10が担当でOK。
- **DAP**：Cayin N7+（クラスA基準。クラスA↔AB切替・ゲイン段あり）。
- **ストリーミング**：Apple Music（メイン）、Qobuz（利用中）。

## 検討経緯で確定した判断

- Kronosの音は試聴で好印象（映画サントラ・オーケストラが特に良かった）。価格ネックだが、同価格帯で他候補も比較検討中。
- **除外/評価が下がった機種**：
  - Phönix・STELLA … ボーカル寄りでVE10と被る＝差分を感じない
  - Macbeth … セミオープンで音漏れ、移動不可
  - MEST Fortune … 試聴でKronosに劣る（外見・ケーブルは良い）
  - DITA Ventura … 自然・実在感型で"包まれる没入"ではない＋遮音弱め
  - Subtonic Storm … 予算超・N7+で鳴らしにくい・入手難
- **条件**：移動でも使う＝密閉・遮音必須／N7+で鳴らせること／予算〜¥693,000。
- 「包まれる没入」が本命軸と判明 → 実質 **Kronos vs Forte Mefisto**（＋装着が合えばSony IER-Z1R）に収束。

## 試聴リストの設計（8カテゴリ）

評価記号 ◎=理想 / ○=良 / △=惜しい / ✕=合わない。
試聴の鉄則：まず深いシール確保（Kronosは左Sサイズ/右Mサイズ）→ 音量は数値でなく聴感で揃える → アッテネーターは付けたまま → 01・02を最優先。

- `00` 装着・シール確認
- `01` ボーカルの艶・生々しさ【最重視】
- `02` キラキラ vs 刺さり【重要・感度】
- `03` 締まったグルーヴ低域（量より質）
- `04` 包まれる没入・空間スケール【決定軸】
- `05` 生音感（実在感・空気・余韻）
- `06` ダイナミクス・強奏
- `07` 解像・分離・一体感
